# ShortenerEngine
간단한 오픈소스 단축기.
## 소개
간단하게 Python Flask로 만든 URL 단축기입니다!
예시는 이 웹사이트에서 보실 수 있습니다. : https://sher.kro.kr/
REPL.IT 의 Replit DB로 작동하며, 간단하게 수정만 하면 바로 작동 시킬 수 있습니다.
